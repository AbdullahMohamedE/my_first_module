# my_first_module
Just a simple task to practice my JavaScript &amp; NodeJS skills. 

This is a simple project to practice what I learned in my JS & NodeJS course.
It is an imitaion of the Math library in JS.
 
##### Module : `ourMath` contains three files
  - `index.js` : where the functions created & exported.
  - `test.js` : applying some simple unit testing to functions.
  - `package.json ` : is the project’s ID file. It stores basic info, startup file, and dependencies.

##### Module : `useMath` uses the `ourMath` module in its `index.js` file. 

##### Skills Practiced: 
  * JavaScript functions and rest parameters
  * Loops and conditionals
  * Arrays and basic array methods (`push`, `includes`)
  * Module system in Node.js (`require`, `module.exports`)
  * Writing reusable functions
  * Basic algorithm design (avg, max, min, set operations)
  * Working with sets logic (union, intersection, difference)
  * Manual testing using `console.log` and `if` checks
  * Problem decomposition (breaking problems into steps)
  * Basic software organization (separating logic into a module)
